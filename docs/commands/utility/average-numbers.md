# Average numbers

## Description

Gets the average of several numbers.

## Aliases

* `average`
* `avg-numbers`
* `avg`
* `averagenumbers`
* `avgnumbers`

## Usage

### Format

`average-numbers <number...>`

### Examples

* `average 55 59 45 61`
* `average 5.01 1.01 -8.04 15.067`

### Arguments

| Name   | Type   | Required |
|--------|--------|----------|
| Number | Number | Yes      |

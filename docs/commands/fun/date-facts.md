# Date facts

## Description

Get a fact about a date

## Details

Not specifying the date to lookup will give you a random fact

## Aliases

* `date-fact`
* `random-date-facts`
* `random-date-fact`
* `datefacts`
* `datefact`
* `randomdatefacts`

## Usage

### Format

`date-facts [month] [day]`

### Examples

* `date-facts`
* `date-facts 46`

### Arguments

| Name  | Type   | Required | Minimum | Maximum |
|-------|--------|----------|---------|---------|
| Month | Number | No       | 1       | 12      |
| Day   | Number | No       | 1       | 31      |

# Year facts

## Description

Get a fact about a year

## Details

Not specifying the year to lookup will give you a random fact

## Aliases

* `year-fact`
* `random-year-facts`
* `random-year-fact`
* `yearfacts`
* `yearfact`
* `randomyearfacts`
* `randomyearfact`

## Usage

### Format

`year-facts [year]`

### Examples

* `year-facts`
* `year-facts 1969`

### Arguments

| Name | Type   | Required |
| ---- | ------ | -------- |
| Year | Number | No       |

# XKCD

## Description

Get an XKCD comic.

## Details

Not specifying the comic to lookup will give you the most recent comic

## Aliases

* `random-xkcd`
* `xkcd-comic`
* `random-xkcd-comic`
* `randomxkcd`
* `xkcdcomic`
* `randomxkcdcomic`

## Usage

### Format

`xkcd [comic]`

### Examples

* `xkcd`
* `xkcd 614`

### Arguments

| Name  | Type    | Required | Minimum |
|-------|---------|----------|---------|
| Comic | Integer | No       | 1       |

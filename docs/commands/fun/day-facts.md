# Day facts

## Description

Get a fact about a day

## Details

Not specifying the day to lookup will give you a random fact

## Aliases

* `day-fact`
* `random-day-facts`
* `random-day-fact`
* `dayfacts`
* `dayfact`
* `randomdayfacts`

## Usage

### Format

`day-facts [day]`

### Examples

* `day-facts`
* `day-facts 14`

### Arguments

| Name | Type   | Required | Minimum | Maximum |
|------|--------|----------|---------|---------|
| Day  | Number | No       | 1       | 31      |

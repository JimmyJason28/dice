# Base64

## Description

Converts text to and from Base64 encoding.

## Modes

`encode` or `decode`

## Aliases

* `base-64`
* `base64`

## Usage

### Format

`base64 <mode> <text>`

### Arguments

| Name | Type   | Required |
|------|--------|----------|
| Mode | String | Yes      |
| Text | String | Yes      |

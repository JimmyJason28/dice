# Color

## Description

Display and convert a color

## Details

Display and convert a color from and to hexadecimal, HSL, RGB, CMYK, and CSS name

## Usage

### Format

`color <color>`

### Examples

* `color blue`
* `color #deaded`
* `color hsl(210,50,50)`

### Arguments

| Name  | Type  | Required |
| ----- | ----- | -------- |
| Color | Text  | Yes      |

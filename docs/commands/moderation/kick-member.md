# Kick member

## Description

Kick a member from your server.

## Aliases

* `kick-user`
* `kick`
* `kickmember`
* `kickuser`

## Usage

### Format

`kick-member <member> [reason for kick]`

### Examples

* `kick @Zoop`
* `kick 213041121700478976`
* `kick Zoop Spamming messages`

### Arguments

| Name            | Type   | Required |
|-----------------|--------|----------|
| Member          | Member | Yes      |
| Reason for kick | Text   | No       |

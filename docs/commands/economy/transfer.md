# Transfer

## Description

Transfer oats to another user.

## Aliases

* `send`
* `pay`

## Usage

### Format

`transfer <amount> <user>`

### Examples

* `transfer 500 @Dice`

### Arguments

| Name   | Type   | Required | Minimum |
|--------|--------|----------|---------|
| Amount | Number | Yes      | 1       |
| User   | User   | Yes      |         |

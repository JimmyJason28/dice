# Face

## Description

Shows an front view of a Minecraft user's face.

## Aliases

* `minecraft-face`
* `mc-face`
* `minecraftface`
* `mcface`

## Usage

### Format

`face <username>`

### Examples

* `face Notch`

### Arguments

| Name     | Type   | Required |
|----------|--------|----------|
| Username | String | Yes      |

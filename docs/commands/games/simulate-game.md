# Simulate game

## Description

Simulate a round of the betting game.

## Aliases

* `practice-game`
* `sim-game`
* `simulate-dice`
* `sim-dice`
* `simulategame`
* `practicegame`
* `simgame`
* `simulatedice`
* `simdice`

## Usage

### Format

`simulate-game <wager> <multiplier>`

### Examples

* `simulate-game 250 4`
* `sim 23-game 2.01`

### Arguments

| Name       | Type    | Required | Minimum | Maximum |
|------------|---------|----------|---------|---------|
| Wager      | Integer | Yes      | 1       |         |
| Multiplier | Number  | Yes      | 1.01    | 100     |

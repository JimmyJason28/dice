# Fortnite statistics

## Description

Get statistics of a Fortnite player

## Platforms

| Name                | Keyword |
|---------------------|---------|
| PC                  | `pc`    |
| Xbox Live           | `xbl`   |
| PlayStation Network | `psn`   |

## Aliases

* `fortnite-stats`
* `fortnite`
* `fortnitestatistics`
* `fortnitestats`

## Usage

### Format

`fortnite-statistics <platform> <username>`

### Examples

* `fortnite-statistics pc Zaccubus`
* `fortnite-stats pc "WBG Strafesh0t"`

### Arguments

| Name     | Type | Required |
|----------|------|----------|
| Platform | String | No       |
| Username | String | Yes      |
